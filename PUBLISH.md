# Как опубликовать игру в интернете

Три шага: залить код на GitHub → бесплатно опубликовать через Vercel → (по желанию) купить домен и подключить.

---

## Часть 1. Залить проект на GitHub

GitHub — это облачное хранилище кода. Оно нужно, чтобы сайт обновлялся сам при каждом изменении.

**Самый простой способ — GitHub Desktop (программа с кнопками, без команд):**

1. Зарегистрируйся на https://github.com
2. Скачай и установи **GitHub Desktop**: https://desktop.github.com
3. В программе: `File → Add Local Repository` → выбери папку `archive-alyh-pisem`.
   - Если предложит «create a repository here» — соглашайся.
4. Внизу слева напиши описание в поле Summary (например `Первый коммит`) → кнопка **Commit to main**.
5. Вверху нажми **Publish repository**. Можешь поставить галочку **Keep this code private** (приватный репозиторий).

Готово — код в облаке. В будущем после правок: Commit → Push.

**Если хочешь через терминал (в VS Code):**
```bash
git init
git add .
git commit -m "Первый коммит"
```
Потом создай пустой репозиторий на github.com и выполни команды, которые он покажет (примерно):
```bash
git remote add origin https://github.com/ТВОЙ_ЛОГИН/archive-alyh-pisem.git
git branch -M main
git push -u origin main
```

> Файл `.gitignore` уже есть — папки `node_modules` и `dist` в гит не попадут, и это правильно.

## Часть 2. Опубликовать сайт бесплатно (Vercel)

1. Зайди на https://vercel.com и войди через **Continue with GitHub**.
2. **Add New… → Project** → найди свой репозиторий → **Import**.
3. Vercel сам определит, что это Vite. Настройки по умолчанию подходят:
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. Нажми **Deploy**. Через ~минуту получишь живую ссылку вида
   `archive-of-scarlet-letters.vercel.app`.
5. Теперь при каждом **Push** на GitHub сайт **обновляется автоматически**.

---

## Часть 3. Купить домен и подключить (по желанию)

1. **Купить домен** (например `scarletletters.com`) можно на:
   Namecheap, Cloudflare, Porkbun или Reg.ru.
   Цена обычно ~$10–15 в год за `.com`.
2. В Vercel: открой проект → **Settings → Domains** → впиши свой домен → **Add**.
3. Vercel покажет, какие **DNS-записи** прописать (обычно одна A-запись и/или CNAME).
   Заходишь в панель у регистратора домена → раздел **DNS** → добавляешь записи ровно как показал Vercel.
4. Через несколько минут (иногда до пары часов) домен заработает, HTTPS подключится сам.

---

## Важно

- Сейчас поток сайта: **главная → кнопка → «Coming Soon»**. Это нормальный тизер — можно публиковать уже сейчас.
- Когда игра будет готова, поменяем кнопку, чтобы вела в саму игру (одна строчка в `src/app/App.tsx`).
